# MULTIPLADA FRONTEND

## Getting Started

### Environment Variables
1. create .env file
```bash
    touch .env
    nano .env
```

2. insert the following to .env file
```bash
    DATABASE_URL=
    COINGECKO=
    NEXTAUTH_URL=
    GOOGLE_CLIENT_ID=
    GOOGLE_CLIENT_SECRET=
    JWT_SECRET=
```

### start
```bash
    yarn
    yarn dev
```
